Flow

One-deck game type. 1 deck. No redeal.

Object

Move all cards to the tableau, in four columns with complete sequences from king to ace.

Quick Description

Like Wave Motion, but cards and card sequences can be moved between reserve piles. Like the tableau piles, the reserve stacks are built down by same suit, but only a single card can be used to fill an empty reserve stack.
