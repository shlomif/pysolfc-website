Carre Napoleon

Forty Thieves type. 2 decks. No redeal.

Object

Move all cards to the foundations.

Quick Description

Like Forty Thieves, but with twelve piles, and during the initial deal, any card that can be moved to the foundations is moved there immediately after each row is dealt. Also, only kings can fill empty piles.
