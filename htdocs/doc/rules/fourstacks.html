Four Stacks

One-deck game type. 1 deck. No redeal.

Object

Arrange all cards in four columns with complete sequences built down in alternate color from king to ace.

Rules

Cards are dealt to four stacks of thirteen cards each, with six empty tableau piles. Piles are built down by alternate color, and any card or valid sequence of cards can be moved between tableau piles. Empty piles can be filled with any card or sequence. The game is won if all the cards can be arranged in four alternate color sequences that build down from king to ace.
