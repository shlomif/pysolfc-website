Relaxed Pyramid

Pairing game type. 1 deck. 2 redeals.

Object

Remove all cards from the Pyramid.

Quick Description

Just like Pyramid, but you win as soon as the Pyramid is cleared.

Notes

Quickplay is disabled for this game.
