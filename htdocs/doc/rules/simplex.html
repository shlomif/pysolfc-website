Simplex

One-Deck game type. 1 deck. No redeal.

Object

Move all the cards to the foundation.

Rules

Nine cards are dealt to nine tableau piles. Tableau piles are built by the same rank. Once all four cards of the same rank are in a tableau pile, that pile may be moved to the foundation. Any card can be moved to an empty tableau pile.

Cards can be dealt from the stock one at a time. No redeal is allowed. The game is won if all cards are moved to the foundation.
