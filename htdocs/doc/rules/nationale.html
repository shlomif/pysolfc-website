Nationale

Baker's Dozen type. 2 decks. No redeal.

Object

Move all cards to the foundations.

Rules

The decks are dealt into twelve piles of eight cards each. An ace and king of each suit are moved to the foundations. Tableau piles are built up or down by same suit. Only single cards can be moved between tableau piles, and any card can be moved to fill an empty tableau pile.

The foundations that start with aces are built up by same suit, while the foundations that start with kings are built down by same suit. The game is won if all cards are moved to the foundations.
