Escalator

Golf type. 1 deck. No redeal.

Object

Move all cards to the foundations.

Quick Description

Just like Relaxed Golf, only with a Pyramid like layout. Like Elevator, but with all cards visible.

Notes

Autodrop is disabled for this game.
