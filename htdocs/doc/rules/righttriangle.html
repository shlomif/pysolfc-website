Right Triangle

Gypsy type. 2 decks. No redeal.

Object

Move all cards to the foundations.

Quick Description

Like Hypotenuse, but once the talon is empty, it can be used as a reserve. Any single card can be moved to this reserve, and the topmost card from this reserve can be moved to a relevant tableau pile or foundation.
