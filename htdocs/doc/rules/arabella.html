Arabella

Klondike type. 3 decks. Unlimited redeals.

Object

Move all cards to the foundations.

Quick Description

Like Triple Klondike, but tableau piles are built down by rank regardless of suit, though only sequences of the same suit can be moved as a unit (similar to Spider).
