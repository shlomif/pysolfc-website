Not Shisen-Sho

Not Shisen-Sho is a variation of Shisen-Sho.

Object

The object of the game is to remove all tiles from the field.

Quick Description

Not Shisen-Sho is played like Shisen-Sho except pairs of tiles can only be removed if they are in the same line horizontally or vertically, even if there are other tiles between them.
