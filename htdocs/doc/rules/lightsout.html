Lights Out

Lights Out type. One deck. No redeals.

Object

Flip all cards face-down.

Rules

A number of cards are dealt in a square grid, with some face-up and others face-down. Any single card can be flipped - if a card is flipped, all of its adjacent cards horizontally or vertically are flipped as well.

The game is won if every card is flipped face-down at the same time.
