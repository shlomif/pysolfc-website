Relaxed PileOn

One-Deck game type. 1 deck. No redeal.

Object

Rearrange the cards so that each pile contains four cards with the same rank.

Quick Description

Like PileOn, but with a single reserve cell added - a single card can be moved to or from this reserve at any time.
