Pegged

Pegged is a classic puzzle game also known as pegged solitaire. It is traditionally played with pegs on a board rather than cards.

Object

Remove all but one card.

Rules

This is a classic puzzle game. At the start of the game, one card is removed. The remaining cards are removed by jumping over neighbour cards, and the space beyond the neighbour must be empty.

You win when there is only one card left.

Notes

To get awarded for a perfect game the remaining card must be in the position of the initial free space.

The first card can be removed automatically, or chosen by the player and removed manually. This is based on an assist level option that can be set in the Options menu.

Autodrop and Quickplay are disabled for this game.
