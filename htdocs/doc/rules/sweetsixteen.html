Sweet Sixteen

One-Deck game type. 1 deck. No redeal.

Object

Move all cards to the tableau

Rules

Sixteen cards are dealt out, forming sixteen tableau piles. Only single cards can be moved between tableau piles, which are built down by alternate color. Empty piles are immediately filled from the talon - cards cannot be dealt from the talon any other way. The game is won if all cards from the talon have been moved to the tableau.
